## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Rosyvero/pinterest-clone.git
   ```

2. Open `index.html` in your browser, or run a local server:
   ```bash
   python -m http.server 3000
   ```

3. Visit `http://localhost:3000`
